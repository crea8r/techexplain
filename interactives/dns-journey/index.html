The 5 Steps of Every Website Visit

1

You type a name

You type "google.com" in your browser. This is called a domain name ‚Äî a human-friendly name that's easy to remember (much better than memorizing numbers!).

2

Browser asks: "What's the number?"

Your browser doesn't know where "google.com" actually lives. So it asks a DNS Resolver (like calling 411 for directory assistance): "Hey, what's the IP address for google.com?"

3

DNS looks it up

The DNS resolver searches its records (or asks other DNS servers) and finds that google.com = 142.250.80.46. This IP address is like a street address for computers.

4

Browser connects to the server

Now your browser knows the address! It sends a request directly to 142.250.80.46: "Hey server, please send me the homepage!"

5

Server sends the website back

The server responds with HTML, images, and code. Your browser receives it all and renders (draws) the beautiful webpage you see!
