# TechExplain

Interactive visualizations that teach tech concepts through exploration.

## Quick Start

1. Open `index.html` in your browser
2. Click on an interactive to start learning

No build step required - it's just HTML, CSS, and JavaScript.

## Structure

```
techexplain/
├── index.html              # Landing page
├── interactives/           # Individual interactive pages
│   └── agent-loop/         # "How AI Agents Work" interactive
├── css/                    # Shared styles
├── js/                     # Shared JavaScript
└── docs/                   # Documentation
```
